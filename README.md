# NandLogic

## Installation

### Submodules

```bash
git submodule init
git submodule update
```

### Build

For Linux :

```bash
mkdir build
cd build
cmake -G "MinGW Makefiles" ..
make
```

For windows :

```bash
mkdir build
cd build
cmake -G "MinGW Makefiles" ..
make
```
